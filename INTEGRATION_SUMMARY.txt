MIGRATION PATHS:

Path 1: Gradual (Recommended)
  1. Build Go CLI: cd cmd/ruw && make install
  2. Run: ./scripts/migrate.sh
  3. Test: ruw status
  4. Enjoy enhanced tools!

Path 2: No Migration
  - Don't build Go/Python tools
  - Everything still works
  - Uses bash fallback
  - No enhanced features

Path 3: Rollback
  - cp bin/ruw.bash.bak bin/ruw
  - mv ~/.git-identities.bak ~/.git-identities
  - Everything reverts to old behavior
